プログラム名
log_analysis.py

実行環境
OS:Ubuntu 16.04.6 LTS (WSL)
言語:Python 3.5.5

実行コマンド
$ python log_analysis.py アクセスログファイルの数 アクセスログファイル名1 アクセスログファイル名2...
例
$ python log_analysis.py 2 access1.log access2.log

1.上記のコマンドを実行します。
2.ファイルの数、ファイル名が不正の場合は再度入力が促されます。
3.期間を指定するかどうかが質問されます。
4.指定する場合は期間の始点と終点の入力が求められます。不正の場合は再度入力が求められます。
5.リモートホスト毎のアクセス記録と時刻毎のアクセス記録が表示されます。対象の記録が存在しない場合は「アクセス記録はありません。」と表示されます。
6.プログラムが終了します。
